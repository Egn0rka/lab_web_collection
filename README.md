# lab_web_collection
All laboratory work on web programming for the third year of university is stored here.
    Work 1
Topic: Creating WEB pages in HTML
Goal: To familiarize yourself with and study the principles of developing WEB pages in HTML.
Task: Develop two WEB pages in HTML for a commercial enterprise and the author.
    Work 2
Topic: Visualizing HTML WEB pages using CSS
Goal: to familiarize yourself with and study the principles of visualizing WEB pages using CSS.
Task: To develop a CSS version of the HTML WEB pages that were created in the previous laboratory work.
    Work 3
Topic: Visualizing HTML WEB Pages Using CSS
Objective: Learn how to create links to different types of documents,
using different types of URLs, and add graphic elements,
tables, and forms to HTML pages.
Task: Develop a resume web page and a table with a structure based on the example.
