<div>
Topic: JavaScript Basics.<br>
Goal. To acquire skills and abilities in using the basic JavaScript language constructs.<br>
Task 1: Write a script that asks the user for a three-digit number and outputs the second digit of this number. To solve the problem, use the % operator (remainder of division)<br>
Task 2: Write a script that, when the user enters the amount of money in the wallet and the price of one chocolate bar. The program outputs how many chocolate bars the user can buy and how much change he will have left.<br>
Task 3: Write a script that asks for 2 numbers and outputs more of them.<br>
Task 4: Write a script that asks the user for the length of a circle and the perimeter of a square. Determine whether such a circle can fit into the specified square.<br>
